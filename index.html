Campus Tube

Campus Tube

Campus Tube is a website that is developed to provide a platform to watch, share, like, dislike videos among a school, moreover it is like a clone of Youtube.

This is my internship project in this project a Master Admin is present which takes care of all happening, students have to create their account on campus tube which will be approved and authorised by master admin, then students can share their videos on existed channels as well as on their own channels

A Transaction System is also present which eventually calculate the transaction points of students as per their videos actions and they can also redeem them through various ways.

The project's Front End is developed using HTML, CSS, JavaScript & Bootstrap, and Backend is developed using Python, Javascript and Framework like Ajax, Django & Jquery along with MongoDB as database.

Campus is deployed on Heroku and authentication is done through Django Admin, Google-OAuth, Facebook OAuth2.
